Instrucciones para realizar el cuestionario

En el siguiente cuestionario puedes encontrar cinco tipos diferentes de preguntas:

  • Las preguntas con múltiples que no se permite escribir las cuales se observa que disponen de una zona para marcar con forma cuadrado. (Checkbox)
  • Las preguntas con una sola respuesta que no se permite escribir, las cuales se observa que disponen de una zona para marcar con forma circular. (Radio)
  • Las peguntas que sean un cuadrado blanco con la palabra “Respuesta” en su interior se tiene que escribir en el cuadrado en cuestión. (Text)
  • Las preguntas que se indican con la frase “Elegir la respuesta correcta” solamente se permite una respuesta por cuadrado en blanco. (Select)
  • En las preguntas que se indica entre paréntesis la frase “Múltiples respuestas” estos permiten elegir más de una opción mediante la utilización de la tecla Ctrl (windows) y Command (Mac). (Select multiple)

Para comenzar el examen haga clic sobre el icono de con el nombre de "Comenzar examen"
